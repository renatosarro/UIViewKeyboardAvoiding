# UIViewKeyboardAvoiding
A simple way to make your UITextField to be always visible, moving up the UIView when the keyboard appears (firstResponder)

# Pod

Podfile
```
pod "UIViewKeyboardAvoiding"
```

Terminal
```
pod install
```
